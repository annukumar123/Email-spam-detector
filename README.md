# 🛡️ Pro Spam Shield
An AI-powered email classifier built with **Scikit-Learn** and **Streamlit**.

## 🚀 Features
- **Real-time Detection:** Paste an email and get instant results.
- **Confidence Scoring:** See how certain the AI is about the classification.
- **Batch Processing:** Upload a CSV for bulk analysis.
- **Data Visuals:** Interactive charts using Plotly.

## 🛠️ Tech Stack
- **Model:** Random Forest Classifier
- **Vectorization:** TF-IDF
- **Frontend:** Streamlit
- **Environment:** Python 3.13
